## Enterprise Data Warehouse & Analytics Platform (GCP)

A cloud-native enterprise data warehouse built using BigQuery and Dataflow,
supporting batch and real-time analytics.

### Features
- Centralized enterprise data warehouse
- Scalable batch and streaming ETL pipelines
- Cost-optimized BigQuery schema design
- BI dashboards using Looker / Data Studio

### Architecture
Sources → Dataflow → BigQuery → Looker
